# CMC Markets Bermuda — BTC Price Lock (Event Build V6)

V6 adds leaderboard-size control, a bottom dashboard CTA on the registration page, and the V5 returning-user / faster-market-update improvements.

## What changed in V6

- leaderboard now supports a configurable number of rows
- admin panel now includes **Leaderboard rows** so you can set 3, 4, 5, 10, and so on
- dashboard table now includes **Entry Price** with helper text
- dashboard table now includes **P/L** helper text
- live BTC/USD refresh cadence changed to **10 seconds**
- live BTC/USD cards flash green/red when the price moves up or down
- returning users are remembered on the same phone/browser using a saved edit token
- same returning user can re-scan the QR code and update the existing entry instead of starting over
- edit throttle removed so prediction changes can be submitted immediately
- confirmation panel CTA order updated so **View the public dashboard** is first and primary
- registration form now also includes a **View dashboard** button at the bottom

## Updating your live Hostinger VPS deployment

Because your current GitHub repo is nested, upload updated files into:

```text
cmc_btc_price_lock_v4/cmc_btc_price_lock/
```

Then redeploy on the VPS with:

```bash
cd /root/cmc-btc-price-lock
git pull
cd /root/cmc-btc-price-lock/cmc_btc_price_lock_v4/cmc_btc_price_lock
docker compose down
docker compose up -d --build
docker ps
docker logs cmc_btc_price_lock-app-1 --tail 50
docker logs cmc_btc_price_lock-caddy-1 --tail 50
```

## Important notes

- the QR code is generated by the app itself and points to `CMC_PUBLIC_BASE_URL`
- returning-user memory is browser/device based and uses local storage plus a secure edit token
- if you want to start clean on an existing install, delete `data/app.db` before launching
